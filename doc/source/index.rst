==========================
Documentation de PortailVA
==========================

PortailVA est une plateforme dédiée aux associations de l'INSA Lyon, qui met à disposition de ces dernières un certain
nombre d'outils leur facilitant la gestion quotidienne.

La plateforme est développée à l'aide du framework `Django <https://www.djangoproject.com>`_ 1.10 et de `Python <https://www.python.org>`_ 3.5.

`Voir notre dépôt Github <https://github.com/VAINSALyon/portailva>`__
